# 院校数据管理指南

## 快速开始

### 启动管理工具

```bash
cd /root/.openclaw/workspace/gaokao-decision-system/backend/scripts
python3 batch_import.py
```

工具会显示一个交互式菜单，提供6种操作方式。

---

## 使用方法

### 方法1：交互式添加单个院校（推荐新手）

选择菜单选项 `1`，按照提示逐步输入：

```
院校ID: pku
院校全称: 北京大学
省份: 北京
城市: 北京
层次: 1 (985)
招生方式: 4 (强基/统招)
2025年最低分: 690
2025年最低位次: 30
选科: 物理,化学
专业1名称: 计算机科学与技术
专业代码: 080901
学科评估: A+
...
```

### 方法2：批量导入JSON文件

**步骤1**: 准备JSON文件

创建一个新的JSON文件（如 `new_universities.json`），格式参考 `data/universities.json`：

```json
[
  {
    "id": "pku",
    "name": "北京大学",
    "province": "北京",
    "city": "北京",
    "level": "985",
    "admission_mode": "强基/统招",
    "website": "https://www.pku.edu.cn/",
    "min_score_2025": 690,
    "min_rank_2025": 30,
    "subjects_required": ["物理", "化学"],
    "majors": [
      {
        "name": "计算机科学与技术",
        "code": "080901",
        "evaluation": "A+"
      }
    ],
    "advantages": ["顶尖名校", "综合实力第一"],
    "disadvantages": ["录取分数极高"]
  }
]
```

**步骤2**: 运行导入

选择菜单选项 `2`，输入文件路径：
```
JSON文件路径: /root/.openclaw/workspace/gaokao-decision-system/data/new_universities.json
```

### 方法3：快速添加常见985/211（最快）

选择菜单选项 `3`，从预设的常见院校列表中选择，只需输入录取数据即可。

### 方法4：查看所有院校

选择菜单选项 `4`，查看数据库中所有院校的列表。

### 方法5：删除院校

选择菜单选项 `5`，输入院校ID删除。

### 方法6：导出到JSON

选择菜单选项 `6`，将数据库中的所有院校导出为JSON文件，方便备份或编辑。

## 数据字段说明

### 必填字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| id | 字符串 | 院校唯一标识（英文） | "pku" |
| name | 字符串 | 院校全称 | "北京大学" |
| province | 字符串 | 省份 | "北京" |
| level | 字符串 | 院校层次 | "985"/"211"/"双一流"/"普通本科" |
| admission_mode | 字符串 | 招生方式 | "统招"/"强基"/"综合评价"/"强基/统招" |
| min_score_2025 | 整数 | 2025年最低分 | 690 |
| min_rank_2025 | 整数 | 2025年最低位次 | 30 |
| subjects_required | 数组 | 选科要求 | ["物理", "化学"] |
| majors | 数组 | 专业列表 | 见下方 |

### 可选字段

| 字段 | 类型 | 说明 |
|------|------|------|
| city | 字符串 | 城市 |
| website | 字符串 | 官网地址 |
| advantages | 数组 | 优势列表 |
| disadvantages | 数组 | 劣势列表 |

### 专业格式

```json
{
  "name": "计算机科学与技术",
  "code": "080901",
  "evaluation": "A+"
}
```

## 数据来源建议

### 获取2025年云南录取数据

1. **官方渠道**（最准确）
   - 云南省招生考试院官网
   - 各高校招生网

2. **第三方平台**
   - 阳光高考网
   - 各省教育考试院

3. **数据格式**
   - 需要：院校名称、最低分、最低位次
   - 注意：确保是云南省物理类数据

### 专业评估等级

- 来源：教育部学科评估结果
- 等级：A+、A、A-、B+、B、B-、C+、C、C-
- 新兴专业可填：快速发展中、重点发展

## 常见问题

### Q1: 院校ID如何命名？
A: 建议使用院校英文缩写（小写），如：
- 北京大学 → pku
- 清华大学 → thu
- 浙江大学 → zju

### Q2: 如果没有2025年数据怎么办？
A: 可以使用2024年数据作为参考，但要在院校名称或备注中标注。

### Q3: 添加错了怎么办？
A: 使用菜单选项 `5` 删除，然后重新添加。

### Q4: 如何备份数据？
A: 使用菜单选项 `6` 导出JSON，或直接备份 `data/gaokao.db` 文件。

### Q5: 导入后前端看不到新数据？
A: 需要重启后端服务：
```bash
# 停止旧进程
kill $(cat /tmp/backend.pid)
# 启动新进程
cd backend && python3 main.py
```

## 数据质量检查

添加数据后，建议检查：

1. **位次合理性**: 985 < 211 < 双一流 < 普通本科
2. **分数位次对应**: 参考2025年一分一段表
3. **专业完整性**: 至少包含1-2个热门专业
4. **选科要求**: 物理类院校通常要求物理
